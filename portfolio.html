I have found my passion in creating, and am always striving to learn all I can in this vast field.

MyFace

End-to-end social media application built in Ruby on Rails. Facebook clone, allowing users to create a profile, post status updates, connect with friends, write on each others' walls, 'like'/'unlike' things, and view a news feed of their friends' activities.

Allergent

End-to-end data analysis application. Allows a user to create a profile, enter in meals (including ingredients) they have eaten. Upon a food intolerance reaction, they can select the type of reaction and severity which then queries the database for the most recent meal eaten and assigns a reaction to its associated ingredients. The User can analyze the data via a table of all ingredients consumed by the user and their frequency for causing reactions, allowing the user to pinpoint what exactly makes them sick. Worked in team of two.

DinoFinder

First full stack application, built in team of three, provided for fundamental understanding of using Github. E-commerce store where users can browse/purchase pet dinosaurs along with any supplies they may require. Related items appear in searches of other items. Users can track past purchases if they have an account.
